# Get Started

Create a workspace:

```bash copy
devmy new
```

Run the application:

```bash copy
cd [WORKSPACE_NAME]
pnpm run dev
```

### Commands

| Command  | Description     | alias |
| -------- | ------- | ----- |
| new      | Create an NX monorepo to scaffold the project | n |
| generate | Generate a new component such as Applications, Add-ons, Libraries  | g |
| --help | Prints information about the command  | -h |
